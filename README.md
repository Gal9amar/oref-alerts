# 🚨 בוט אזעקות פיקוד העורף — טלגרם

בוט טלגרם שמושך נתונים מפיקוד העורף ומשלח התראות בזמן אמת.

## פקודות

| פקודה | תיאור |
|-------|--------|
| `/start` | הרשמה לקבלת התראות |
| `/stop` | הפסקת התראות |
| `/now` | בדיקת אזעקות פעילות כרגע |
| `/stats` | סטטיסטיקות 24 שעות לפי עיר |
| `/city [עיר]` | חיפוש אזעקות לפי עיר |

## פריסה על Railway

1. חבר ריפו ל-Railway
2. הוסף Environment Variable:
   ```
   BOT_TOKEN = <הטוקן שלך>
   ```
3. פרוס — הבוט יתחיל לרוץ אוטומטית ✅

## הערות

- הבוט מאזין לאזעקות כל **2 שניות**
- עובד רק מ-IP ישראלי (Railway אירופה קרוב מספיק)
- אין צורך ב-Netlify או שרת נפרד
